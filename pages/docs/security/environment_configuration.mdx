# Environment Configuration

## Overview
The application uses environment-specific configuration for security settings and features.

## Production Environment
```env
DEBUG=False
DJANGO_ENV=production
CSRF_COOKIE_SECURE=True
CSRF_COOKIE_SAMESITE=Strict
SESSION_COOKIE_SECURE=True
SECURE_SSL_REDIRECT=True
```

## Development Environment
```env
DEBUG=True
DJANGO_ENV=local
CSRF_COOKIE_SECURE=False
CSRF_COOKIE_SAMESITE=Lax
```

## Security Variables
- `SECRET_KEY`: Application secret key
- `FIELD_ENCRYPTION_KEY`: Data encryption key
- `JWT_ACCESS_TOKEN_LIFETIME`: Token validity period
- `CSRF_URL_SUFFIX`: CSRF endpoint suffix
